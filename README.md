# Windows-custom

Nơi này để lưu trữ 99,9% các file .ini, .rmskin và các tệp (có liên quan) để custom Windows. Tôi là người thích chỉnh sửa giao diện (theo vibe) nên tạo thư mục này để lưu trữ các file.
